React project setup:
--------------------


Node: Runtime for build tools (V8)
Webpack: Build automation tools
npm: node package manager
package.json: configuration file for project used by npm.
ES6 : ECMA6 - 18 new features (cross browser)
Babel: Is a transpiler which converts ES6 to ES5.
Webpack-bable plugins (Balel with webpack)

React: View library.
React-dom     reactjs.org

npm install react react-dom --save

Debugging: react-dev-tool (chrome extension)

Redux: State manager
redux-dev-tool debugger (chrome extension)

Responsive design: bootstrap (outdated)
Material design: Rich UI targetted for all devides - By Google.

plugins for deployment: uglify
